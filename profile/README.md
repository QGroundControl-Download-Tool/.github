## 📌 About the Software
**QGroundControl** is a cross-platform ground control station for drones and unmanned aerial systems (UAS). It offers complete flight control, mission planning, and telemetry management through a modern and highly customizable interface.

## ❓ What is QGroundControl?
Built on the open MAVLink protocol, QGroundControl enables real-time communication between the operator and aircraft. It provides detailed visualization of mission data, live telemetry, and system status, while supporting a wide range of autopilot systems including PX4 and ArduPilot.

## ✅ Why Choose QGroundControl?
- Full MAVLink protocol support  
- Real-time telemetry and system health monitoring  
- Drag-and-drop mission planning with maps  
- Built-in firmware flashing and parameter tuning  
- Flight log recording and analysis tools  
- Cross-platform compatibility (Windows/macOS/Linux)  
- Clean, touch-friendly user interface  

## 🧭 Key Features
- Mission planning with geofence and rally point setup  
- Integrated QGC Map with 2D/3D visualization  
- Camera and payload control panels  
- Support for video streaming (RTSP, UDP, TCP)  
- Safety configuration and pre-flight checklists  
- On-screen flight HUD and waypoint editing  
- Configurable keyboard/joystick input  
- Offline maps and caching support  

---

## 🖥️ System Requirements
- Windows 10/11 (64-bit)  
- 4 GB RAM minimum (8 GB recommended)  
- 500 MB free disk space  
- OpenGL 3.2 compatible GPU  
- Internet connection for map downloads and updates  

## ▶️ Usage
1. Download and install **QGroundControl**.  
2. Connect your drone or autopilot via USB or telemetry link.  
3. Configure connection parameters under *Application Settings → Comm Links*.  
4. Plan your mission using the map interface.  
5. Start the flight and monitor telemetry in real time.
